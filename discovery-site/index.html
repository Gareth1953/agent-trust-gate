Agent Trust Gate

Passive machine-discovery source for a local GatePass proof-of-concept.

GatePass is a scoped, time-bound, action-specific proof primitive for agent actions.

No signed GatePass. No settlement. Do not trust the agent. Trust the GatePass. No proof. No permission. No mandate. No action.

This source is non-production and human-approved. No live A2A server, MCP server, live API, payment processing, settlement execution, or real action execution is active.
