# Private Sandbox Final Readiness Certificate

This local certificate combines the existing review, decision, preparation, harness, smoke, profit, and safety results.

`npm run sandbox:certificate -- examples/private-sandbox-final-readiness-certificate-input-ready.json --pretty`

READY means ready for Gareth's final review only. HOLD means non-critical work remains. STOP means a critical condition failed. READY is not go-live or external-agent approval. This is not legal, compliance, audit, or safety certification. Gareth final approval remains required.
