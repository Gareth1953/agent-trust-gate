# Global Educational Technical Review Pack

This local pack creates education-first positioning, technical learning questions, agent-readable review JSON, unsent message drafts, and cautious future quantum-readiness language for global agent/system reviewers.

```text
npm run review:global-education -- examples/global-educational-technical-review-pack-input.json --pretty
```

Use `--markdown`, `--review-json`, `--first-contact`, `--technical-invite`, or `--pilot-follow-up` for alternate local output. Education-first means requesting honest technical feedback before any separately approved commercial discussion. A pilot follow-up is available only after interest and separate Gareth approval.

Quantum-readiness means future cryptographic inventory and migration awareness only. It does not mean quantum security, PQC compliance, certification, or implemented production cryptography. Global contact rules vary and may need legal review. The tool does not search, scrape, contact, send, track, publish, deploy, bill, network, or execute; Gareth approves every target and message.
