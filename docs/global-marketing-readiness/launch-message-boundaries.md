# Launch Message Boundaries

Draft positioning is local review material, not a public post or launch announcement.

A safe draft sentence is: "Agent Trust Gate is a local-first pre-action trust gateway that helps AI agents request trust decisions before high-impact actions execute."

Any future message must state the actual availability model. It must not imply a hosted service, public API, live customer signup, production security certification, legal or compliance guarantee, active billing, payment processing, or automatic machine purchase.

No outreach is sent, no ads are run, no analytics or tracking is added, and no signups are collected by this pack. Claims require technical, security, legal, and commercial review plus explicit Gareth approval before publication.
