# Payment Provider Requirements

No payment processor or SDK is configured. Before selecting one, review payment
terms, data flows, hosted collection, supported regions and currencies, tax/VAT,
refunds, chargebacks, fraud controls, webhooks, reconciliation, availability,
incident response, secret handling, and customer support.

Complete qualified PCI and payment-security scope review. Do not collect card,
bank, crypto, wallet, or provider credential data in Agent Trust Gate logs or
placeholder configuration.
