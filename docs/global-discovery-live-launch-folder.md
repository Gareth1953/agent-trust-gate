# Global Discovery Live Launch Folder

This builder creates the first upload-ready global discovery step: 15 static files for human, LLM, developer-tool, and future agent review. It exposes no engine, backend, API, or live endpoint.

Run `npm run launch:global-discovery -- examples/global-discovery-live-launch-folder-input.json --pretty`. Add `--dry-run` to avoid writes or `--list-files` to list the fixed bundle.

Upload-ready means locally generated and reviewable, not published. Gareth approval is required before upload or domain connection. Payment, billing, settlement, tracking, analytics, real agents, action execution, and quantum-security claims remain disabled.
