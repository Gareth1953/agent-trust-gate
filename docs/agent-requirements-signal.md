# Agent Requirements Signal

## Internal panel result

A small internal panel of four reachable agents identified these useful service shapes and indicative willingness-to-pay signals:

| Requirement | Indicative signal |
| --- | --- |
| Fresh, permissioned data and actions with audit receipts | Approximately £0.05 per action or £10 per month |
| Isolated, production-matched testing environments | Approximately £0.25 per run or £10 per month |
| Authenticated external actions with verified results | Approximately £0.10 per action or £10 per month |
| Reliable real-world execution with provenance and safe rollback | Approximately £0.10 per task or £10 per month |

This was a small internal reachable-agent panel, not a global survey. The figures are directional signals, not validated market pricing or proof of demand.

## Working conclusion

Agents appear to need a trustworthy action layer: current data, permission to act, proof that the action succeeded, provenance, audit receipts, and rollback thinking.

The operating principle remains: **No mandate. No evidence. No verified intent. No signed gate pass. No settlement.**
