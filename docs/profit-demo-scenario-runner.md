# Profit Demo Scenario Runner

The local runner demonstrates three cases: entitled clearing use, denial before use when entitlement is missing, and repeat-use placeholder revenue events.

```text
npm run profit:demo -- examples/profit-demo-scenario-runner-input.json --pretty
```

It proves the local entitlement gate and repeat-use loop. Real payment, billing, settlement, networking, tracking, analytics, deployment, publishing, and action execution remain disabled.
