# Private Sandbox Decision Gate

This local gate checks profit-demo, controlled-sandbox, smoke-test, entitlement, denial, safety, and approval readiness.

`npm run sandbox:decision -- examples/private-sandbox-decision-gate-input-ready.json --pretty`

“Ready” means ready for Gareth's review only. It is not permission for external testing and is not legal, compliance, audit, or safety certification. Network, public API, agents, payment, billing, settlement, tracking, analytics, deployment, publishing, and action execution remain disabled. Gareth final approval is required before any private external test.
