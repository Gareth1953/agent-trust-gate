# Reviewer Demo Limitations And Safety Boundary

## Local Only

The one-command reviewer kit is local-only. It runs deterministic local models and reads no live external systems.

## Mock Tool Execution Only

The developer wrapper portion may call local mock functions after valid proof. It does not execute real tools, real emails, real customer messages, real procurement, real payment preparation, wallet actions, banking actions, settlement actions, network calls, or browser automation.

## Not Production Middleware

The reviewer kit is not production middleware and does not claim production readiness. It is a proof-of-concept reviewer path for local inspection.

## Not Benchmarking Or Certification

The scorecard timing is local illustrative timing only. It is not a production benchmark, not security certification, not adversarial completeness, and not legal, compliance, procurement, settlement, identity, authentication, or security assurance.

## Not A Live Framework Integration

The local framework-style example is framework-inspired only. It is not a real LangGraph integration, has no LangGraph dependency, makes no external network call, and executes no live agent framework.

## No Live Authority

The reviewer kit adds no live API, MCP server functionality, hosted service, live systems contact, direct bot messaging, live agent-to-agent communication, external-agent contact, autonomous contact, outreach automation, payment processing, settlement execution, production signing, production key management, secrets, credentials, or action execution.

## Core Boundary

GatePass is a scoped, time-bound, action-specific proof primitive for agent actions.

No signed GatePass. No settlement.

Do not trust the agent. Trust the GatePass.

No proof. No permission.

No mandate. No action.
