# Fast Reaction Trust Gate

Fast Reaction Trust Gate is a future optional enhancement, not an immediate build.

Inference speed is the time needed to evaluate trust inputs and choose a decision. External task latency is the time consumed by tools, services, networks, approvals, or real-world execution. Improving one does not remove constraints in the other.

Speed must never override trust.

## Future routing tiers

- Instant allow for low-risk, trusted, repeatable actions.
- Gated allow for medium-risk actions with fresh evidence.
- Human review for high-risk, money, legal, or customer-impacting actions.
- Instant refuse for unsafe, no-mandate, stale, or over-budget actions.

**Agent Trust Gate™ does not just decide if an agent may act — it decides how fast it may safely act.**

No routing tier authorises live action, payment, or settlement in the current project.
