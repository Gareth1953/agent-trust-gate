# Global Agent Outreach Plan

Global outreach should begin only after the core Agent Trust Gate™ is safe, clean, tested, documented, and near completion.

## Sequence

1. Finish the core local build.
2. Prepare a public developer pack.
3. Prepare an agent-readable capability statement.
4. Prepare a safe sandbox invitation.
5. Conduct controlled outreach to agents, agent systems, MCP/A2A-style builders, payment protocol builders, enterprise automation teams, procurement-agent systems, and machine-to-machine infrastructure groups.

## Core message

“Before an agent acts, spends, delegates, or settles, it needs proof: valid mandate, verified intent, trusted evidence, spend limits, audit receipts, and a signed gate pass.”

This plan does not authorise automated outreach, scraping, sending, live agent contact, payment, or production integration. Every future contact remains subject to Gareth approval.
