## Suggested longer public summary

Agent Trust Gate™ is a local-first pre-action / pre-settlement trust
enforcement layer for agent-led actions and payments. It demonstrates how a
proposed agent action can be checked for mandate, evidence, verified intent,
policy/risk tier, receipt validity, replay safety, and settlement-blocker
eligibility before any simulated settlement decision is allowed.

Core line:

**No mandate. No evidence. No verified intent. No signed gate pass. No settlement.**

This repository is a public, code-readable local developer proof. It contains
TypeScript source, deterministic local tests, synthetic examples, JSON schemas,
a static manifest, documentation, CLI proof commands, launch records, and a
static developer landing page file for review.

Developers, agent builders, and integration reviewers can clone the repository,
install dependencies, run the tests, run the local demos, inspect the schemas
