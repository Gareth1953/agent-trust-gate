# Pilot Conversion Path

P3-M145 supports a responsible commercial progression from local proof to scoped integration discussion.

## Progression

1. Local technical demonstration.
2. Customer-specific feasibility review using synthetic or sanitised inputs.
3. Integration design.
4. Controlled non-production adapter.
5. Separately authorised production engineering.
6. Ongoing policy and evidence operation.

## Local Technical Demonstration

The local demonstration proves the trust-control journey:

- proposed action;
- mandate check;
- scope check;
- spend or value limit check;
- evidence check;
- approval check;
- GatePass or refusal receipt;
- simulated settlement allowed or blocked;
- local audit evidence saved.

## Feasibility Review

A customer-specific review can map the customer's synthetic or sanitised action categories, policies, limits, approval requirements and evidence model into the local GatePass pattern.

The review remains non-production and commercially agreed.

## Integration Design

Integration design may describe how Agent Trust Gate would fit beside the customer's agent platform, policy store, approval workflow, audit store, payment-adjacent system or execution environment.

It does not itself create live payment, settlement, runtime enforcement, SSO, secret storage or production isolation.

## Controlled Non-Production Adapter

If separately scoped, a non-production adapter could connect synthetic or test systems to the GatePass decision flow. This requires written scope, security review, data review and human approval.

## Production Engineering

Production engineering is separate. Completing the local pilot does not automatically create production readiness, production deployment, production signing, legal compliance, payment protection, settlement authority or autonomous execution.

## Ongoing Operation

If production work is later authorised, ongoing operation would need policy ownership, evidence freshness rules, approval governance, replay and expiry management, monitoring, incident response and integration controls outside this local repository.
