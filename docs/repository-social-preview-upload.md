# Repository Social Preview Upload

GitHub social preview should be updated manually through GitHub if Gareth chooses to do so.

Manual route:

Settings -> General -> Social preview -> Edit -> Upload an image

Use the previously approved Agent Trust Gate image slate manually. Do not create, edit, generate, upload, or automate any image from this repository mission.

## Image Requirements

Current suitable GitHub social preview image requirements:

- PNG, JPG, or GIF;
- under 1 MB;
- at least 640 x 320 pixels;
- preferably 1280 x 640 pixels;
- solid background preferred for consistent display.

## Boundary

This document is a manual route only. It adds no image-generation step, no upload automation, no external service call, no tracking, no analytics, and no public announcement automation. X remains the only approved public announcement channel.
