# Product Overview

AI agents can propose public communications, customer commitments, spending,
money movement, and compliance-sensitive work. Agent Trust Gate places a
deterministic pre-action decision between a proposal and the caller's execution.

- `ALLOW`: local rules allow the exact described action.
- `BLOCK`: the caller must stop.
- `REQUEST HUMAN`: explicit review is required before the caller proceeds.

Receipts, approval packs, review records, and evidence bundles provide a portable
local audit trail. They record inputs and decisions; they do not prove truth,
identity, authorization, legality, compliance, or safety. Agent Trust Gate does
not execute actions or authenticate real-world identities.
