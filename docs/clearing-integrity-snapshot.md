# Clearing Integrity Snapshot

The snapshot combines allowlisted count summaries from the local clearing ledger, RefusalGraph signal store, evidence bundles, replay runs, receipt verification, fee placeholders, and batch clearing. It provides a deterministic local demo integrity indicator for future machine-to-machine clearing review.

This is local-only and draft-only. It is not live monitoring, user analytics, tracking, legal evidence, compliance or safety certification, a live audit, public reporting, network verification, or blockchain proof. It writes nothing and performs no billing, payment, settlement, deployment, publishing, outreach, webhook, third-party connection, or action execution.

The score uses aggregate mismatch, failure, incomplete-evidence, missing-receipt, and safety-mismatch counts only. Private customer, company, account, wallet, credential, document, endpoint, URL, and message fields are ignored.

Any live monitoring, external verification, public reporting, commercial, or production use requires technical, security, privacy, legal, audit, publication, and commercial review plus Gareth final approval.
