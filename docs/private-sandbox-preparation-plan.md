# Private Sandbox Preparation Plan

This local plan defines scope, fake/test agents, prohibited capabilities, boundaries, approvals, and stop conditions required before any private sandbox test can be considered.

`npm run sandbox:plan -- examples/private-sandbox-preparation-plan-input-ready.json --pretty`

`preparation_ready` means the written plan is ready for Gareth review. It is not go-live, deployment, publication, external-agent, network, payment, or action approval. All live capabilities remain disabled and Gareth final approval is required.
